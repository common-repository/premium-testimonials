== Description ==

<h3>Premium Testimonials</h3>

This plugin will help your visitors decide to buy your services or products.
Easily integrate our plugin as a widget in your sidebar or use a shortcode to display testimonials in multiple ways.

Demo: [Live Preview](http://demo.eztiv.com/?theme=EZ%20Premium%20Testimonials)

Top Features:

1. Clean code
1. Easy to use
1. Modal Popup submission form
1. Ajax/jQuery Form Submission
1. Powerful admin tools for managing the testimonials
1. Sidebar widget with two viewing options: list or cycle
1. Shortcode to embed testimonial form
1. Shortcodes for multiple viewing with custom options (Portraits / Boxed / Masonry / Jquery Slider (carousel) / List)

== Installation ==

1. Upload the entire contents of the zip file to your plugin directory '/wp-content/plugins/'
1. Activate the plugin through the 'Plugins' menu in WordPress
